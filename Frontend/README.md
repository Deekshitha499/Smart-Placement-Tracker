# 🎨 Smart Placement Tracker - Frontend

Frontend for Smart Placement Tracker developed using React.js and Tailwind CSS.

This platform provides separate dashboards for Students and TPOs with modern UI/UX and AI-powered placement preparation features.

---

# 🚀 Features

# 👨‍🎓 Student Features

- Student Login & Registration
- Student Dashboard
- Companies Section
- Placement Drives
- My Applications
- ATS Resume Checker
- Mock Interviews
- Aptitude & Technical Tests
- Notifications
- Profile Section
- Logout Functionality

---

# 🏢 TPO Features

- TPO Login & Registration
- Dashboard Analytics
- Company Management
- Student Management
- Application Tracking
- Notifications
- Profile Section

---

- Tailwind CSS

# ⚙️ Installation

# 1️⃣ Install Dependencies

```bash

```

---

# 2️⃣ Start Frontend

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

# 🚀 Smart Placement Tracker

A full-stack AI-powered placement management platform developed for colleges and universities to efficiently manage placement drives, student applications, mock interviews, ATS resume analysis, tests, and recruitment activities.

---

# 📌 Project Overview

Smart Placement Tracker is a modern web application that helps:

- 🎓 Students prepare for placements
- 🏢 TPOs manage recruitment drives
- 📄 Analyze resumes using ATS checker
- 🧠 Practice mock interviews
- 📝 Attend technical & aptitude tests
- 📊 Track applications and placement statistics

The platform provides role-based dashboards for both Students and TPOs with a professional UI and interactive features.

# ⚙️ Installation

# 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/smart-placement-tracker.git
```

---

# 2️⃣ Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

# 3️⃣ Backend Setup

```bash
cd Backend
npm install
npm start
```

Backend runs on:

```bash
http://localhost:5000
```

---

# 🔐 Environment Variables

Create `.env` inside Backend:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
